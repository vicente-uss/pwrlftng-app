# PWRLFTNG

App móvil de registro de entrenamiento (training log) para powerlifting. Registra tus series, pesos, reps y RPE/RIR de forma rápida y minimalista, y consulta tu historial y récords personales — sin coaching automatizado, solo los datos para que tú o tu coach tomen las mejores decisiones.
