# Roadmap PWRLFTNG

## Semana 1 — Base navegable
- Proyecto Expo + TypeScript.
- Login demo.
- Tabs inferiores.
- Home, Entrenar, Perfil.
- Rutinas y creación básica.
- Sesión activa demo.

## Semana 2 — Persistencia local
- SQLite.
- Guardar rutinas reales.
- Guardar sesión activa.
- Recuperar sesión si se cierra la app.
- Editar series en rutina.

## Semana 3 — Motor de entrenamiento
- Agregar ejercicios durante sesión.
- Editar peso/reps/RPE/RIR en vivo.
- Historial por ejercicio.
- Última vez por ejercicio.
- Mejor peso y más reps con mismo peso.

## Semana 4 — Beta privada
- Pulido visual.
- Corrección de errores.
- Documentación para coach.
- Demo funcional.
- Preparación para build Android interna.

## Fase 2
- Supabase Auth.
- Sincronización nube.
- Login Google.
- Panel web coach.
- Importación Excel.
- Comentarios del coach.
- Gráficos de volumen semanal y grupo muscular.
