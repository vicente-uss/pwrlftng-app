# Sprint 01 — Demo navegable

## Objetivo
Crear la primera versión funcional y navegable de PWRLFTNG para validar el flujo con atleta y coach.

## Alcance técnico
- Expo + React Native + TypeScript.
- Expo Router para navegación.
- Modo oscuro premium.
- Login demo con cualquier credencial.
- Home, Entrenar, Perfil.
- Rutinas, Crear Rutina, Detalle Rutina.
- Sesión activa con cronómetro y descanso automático.
- Resumen final básico.
- Datos temporales en memoria.

## Fuera de este sprint
- SQLite real.
- Supabase Auth.
- Login Google.
- Importación Excel.
- Panel coach.
- Gráficos avanzados.

## Criterio de aceptación
El beta tester puede abrir la app, entrar con login demo, ver rutinas, crear una rutina, iniciar entrenamiento, marcar series, activar descanso y finalizar sesión con resumen.
